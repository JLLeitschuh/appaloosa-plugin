Names of the groups, separated by "|" that will allowed to see this version. Example: "Group 1 | Group 2". When empty the update will be publish to previous allowed groups if a previous update exists, otherwise it will be published to default group "everybody". You can also specify to publish your file to the default group "everybody", you have to use the name "everybody" (even if your store is in French). See available environment variables here. Example: $GIT_COMMIT
